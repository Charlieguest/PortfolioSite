Projects

Below is a list of projects I've worked on and am proud of. I've recently took part in various Game Jams which have allowed me to develop my skills and practice working in a time constrained environment.

Titan's Roots

Titan's Roots is an 8-bit game demo developed for the Global Gam Jam. We used the unity engine to create a survival mini game where you are a plant's looking around for nutrients.

The Underlight

The underlight is another game demo. A longer more drawn out 1st person experience which was made for the Brackey's Game Jam. We had 7 days to design a game around diving deeper.

We decided to create a puzzle mystery game where you are exploring the unexplained disappearances outside a mansion in the woods. We encorparated three levels that unfolded this mystery

We used github for collaborative working which created some issues so we didn't manage to complete the whole project. We did get a fair amount done and this project was submitted via Itch.
